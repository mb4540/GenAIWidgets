# Application Style Guide

## Overview
Update the full application style to be modern, accessible, and visually engaging.

## Requirements

### 1. Theme Support
- **Light/Dark Mode**: Full support for both light and dark color schemes.
- **Toggle**: Accessible switch to toggle between modes (auto-detect preference by default).

### 2. Color Palette
- **Accents**: Incorporate colorful accents to add vibrancy and guide user attention.
- **Semantic Colors**: Define clear colors for success, error, warning, and info states.
- **Contrast**: Ensure WCAG compliance for text and interactive elements.

### 3. Iconography
- **Library**: Establish a consistent iconography library (e.g., Lucide React).
- **Usage**: Consistent sizing, stroke width, and coloring throughout the application.
